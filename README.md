# e-plantShopping

e-plantShopping is a React-based online plant shopping application built using
React, Redux Toolkit, and React Router.

## Features
- Landing page with background image and “Get Started” button
- Product listing grouped into multiple plant categories
- Add-to-cart functionality using Redux Toolkit
- Shopping cart with quantity updates and total price calculation
- Reusable components and clean UI design

## Technologies Used
- React
- Redux Toolkit
- React Router DOM
- HTML5 & CSS3
